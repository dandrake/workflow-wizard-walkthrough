Everyone: first, go to https://adoptium.net.

Mac (Intel and Apple Silicon):

  1. The Adoptium homepage will have a big download button. It usually offers the installer for the correct processor.

    ⚠️ Double check the offered installer: Intel Macs should get the "x64" version, and Apple Silicon Macs should get the "aarch64" version.

    If the site offers the wrong installer:
    1. Select "Other Downloads" then scroll down the page to where you can choose your processor. Select yours: "aarch64" for Apple Silicon, "x64" for Intel.
    2. Make sure the correct one is highlighted and download the .PKG installer.

  2. Open/run the installer when it is done downloading.

    If you get a message about having to install Rosetta, then you are using an Apple Silicon machine and you downloaded the wrong version. Go back to the previous step and ensure you download the aarch64 installer.

  3. Go through the installation process accepting all the defaults.

Windows:

Go through the installation process accepting all the defaults.

  1. The installer may ask to “make changes to your device.” If it does, click Yes.
  2. Make sure to set the JAVA_HOME variable by clicking the dropdown and selecting the first option as shown in blue below.
  3. This is the folder area to find where the JDK was installed: “Eclipse Adoptium”

If the installation succeeds, then…nothing happens. Don’t worry! If you didn’t get any errors, it probably worked — and if you find out later on that it didn’t, ask for help and we’ll fix it together.

Removing configuration for older versions of Java:
  1. Open up Powershell by tapping the Windows key and typing powershell
  2. In Powershell, enter cd ~ and hit enter then type explorer . and hit enter.
  3. Delete the hidden folder .gradle .
  4. Restart your computer.
