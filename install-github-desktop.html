1. Go to https://desktop.github.com/download/

  2. You'll likely see something like the screenshot below. Download the version appropriate for your processor.

  3. Install and launch GitHub Desktop when it is done downloading.
  4. To keep things organized, you should move the downloaded file out of your Downloads folder and into Applications. Note, you might need to uncompress the downloaded file.

  5. You should see a welcome message like the screenshot below. Click the Sign in to GitHub.com button:

  6. You should get a message asking you to authorize GitHub Desktop. Accept, and it should take you back to a screen in GitHub Desktop that looks like this:

    If you don’t see this screen, and instead it looks like it is still waiting for you to log in, click the Cancel button, and then try the option that says “Sign in to GitHub.com using your username and password” instead.

    If you do see this screen, do enter your real name, but don’t change the email! Click Continue.

  7. Click Finish.
