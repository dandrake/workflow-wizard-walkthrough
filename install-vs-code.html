⚠️ There is a product named “Visual Studio.” There is also a product named “Visual Studio Code.” They are completely different products. Microsoft just gave them very similar names in order to confuse us all. (Thanks, Microsoft!) For this class, we want Visual Studio Code.️
Everyone:
  1. Go to https://code.visualstudio.com/
  2. Click the big Download button.

Mac (Intel and Apple Silicon):

To keep things organized, you should move the downloaded file out of your Downloads folder and into Applications. Note, you might need to uncompress the downloaded file.

Windows:

Double-click the app to open and start configuring VS Code.

  1. Run / open the downloaded file.
  2. Accept the license agreement. Click Next.

  3. You can mostly keep clicking “Next” to accept all the installation defaults, with one exception: make sure you enable the following three options on the “Select Additional Tasks” screen:

    ☑ Add "Open with Code" action to Windows Explorer file context menu
    ☑ Add "Open with Code" action to Windows Explorer directory context menu
    ☑ Register Code as an editor for supported file types
  4. Click the Install button when you get to the end of all the configuration screens.
  5. When the installation is done, make sure that “Launch Visual Studio Code” is checked:

    ...and click Finish.
